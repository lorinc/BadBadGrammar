# Bad Bad Grammar

A research-backed essay arguing that traditional grammar instruction channels language knowledge into the wrong brain system — one incapable of producing fluent speech.

Four decades of converging evidence from neurolinguistics, cognitive science, and classroom research show that explicit grammar teaching builds declarative memory representations (knowing *about* a language) instead of the procedural competence (using a language automatically) that underlies all fluent communication.

The essay presents the foundational and recent evidence, reviews what actually works — comprehensible input, TPRS, extensive reading, immersion — and contextualises findings within the European and Spanish educational landscape, with attention to Montessori pedagogy.
